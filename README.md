## üì¶ Getting Started (Local Setup)

To run NoteMaster on your local machine, follow these simple steps:

1.  **Clone the Repository:**
    Use `git` to download the project files:
    ```bash
    git clone [https://github.com/vinayakgoyal2208/note-master.git](https://github.com/vinayakgoyal2208/note-master.git)
    ```

2.  **Navigate and Open:**
    Change into the project directory and open the files in your code editor:
    ```bash
    cd note-master
    code .  # If using VS Code
    ```

3.  **Run the Application:**
    Locate the `index.html` file and open it directly in your web browser (double-click).
    
    *For best development experience, use the **Live Server** extension in VS Code.*

## üí° How It Works (User Guide)

1.  **Adding a Note:** Click the dedicated button or icon to generate a new note pad.
2.  **Typing:** Start typing immediately. Every keystroke is saved automatically to your browser's local storage.
3.  **Resizing:** Hover over the bottom-right corner of the editor to freely resize the note area.
4.  **Persistence:** You can close your browser tab; when you reopen the app later, your notes will be loaded back exactly as you left them.
